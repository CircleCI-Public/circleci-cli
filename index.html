CircleCI in your
command line.

Trigger and watch runs, inspect workflows and jobs, pull down artifacts, and validate config — all from your terminal. Every data-returning command speaks --json, so it drops straight into scripts and CI.

  • macOS — Homebrew
  • Windows — WinGet
  • Linux — Snap

If you have the stable CLI installed:

$ brew uninstall circleci

Then:

$ brew tap circleci-public/circleci
$ brew trust circleci-public/circleci # for brew > v6
$ brew install circleci@next
$ winget install --id CircleCI.CLI.Preview
$ sudo snap install circleci --channel=edge
$ sudo snap connect circleci:password-manager-service
